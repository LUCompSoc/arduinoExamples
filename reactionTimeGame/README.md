# Reaction Time Game
This is a small little game to test your reaction time.
It uses the display talked about in [README.md](/README.md) to show the user's reaction time.

Your task is to replace the question marks (`?`) with the correct code to make it work!
See slides 20-22 in [Intro to Arduinos.pdf](/Intro%20to%20Arduinos.pdf) for more info, including stretch goals if you want to challenge yourself.

Have fun :>
